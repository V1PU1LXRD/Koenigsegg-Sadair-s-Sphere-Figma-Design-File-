## 🔥 Design Highlights

### ⚡ Performance Specs Featured
- **1625 hp** Power visualization
- **360 km/h** Top speed indicator
- Immersive hypercar 3D-style hero imagery

### 🎨 Design Philosophy
This concept landing page draws inspiration from Koenigsegg's relentless pursuit
of engineering perfection. The dark, moody color palette paired with warm amber
accents creates a sense of power and luxury. Every element — from the typography
hierarchy to the asymmetric layout — is intentional.
